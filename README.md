# NBA Stats Analyzer
A Python data analysis project built using pandas and matplotlib.

## How to Run

1. Install required packages:

```bash
pip install pandas matplotlib
```

2. Download the repository.

3. Run:

```bash
python analyzer.py
```

## Features
- View the top 5 NBA teams by wins
- Compare Eastern and Western Conference scoring averages
- Search for a specific NBA team
- Display a chart of the top teams by wins
- View the highest-scoring teams
- View the teams with the best field goal percentage

## Technologies Used
- Python
- pandas
- matplotlib

## Dataset
This project uses real NBA team statistics from the 2025-26 NBA season.
